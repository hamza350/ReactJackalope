Rates API (All Insurance Companies)

Get top three cheapest rates for the same coverage.

URL

https://www.ninjaworkcomp.com/api/v1.0/rates/{startDate}/{exModFactor}/{classCode}/{payroll}/{zipCode}/{applyFilter}

HTTP Method

GET

Response Type

ListOfRates

Parameters

  • Path: startDate String

    Policy Start Date

  • Path: exModFactor Double

    Experience Modifier Factor

  • Path: classCode Integer

    Insurance Class Code

  • Path: payroll Integer

    Company Payroll

  • Path: zipCode String

    Postal Code

  • Path: applyFilter Boolean

    Apply Company Filter Rules (priority rules)

  • Query: subCode String

    SubCode of Class Code, If Any

  • Query: sicCode Integer

    Sic Code, If Any (relevant for Hartfor and Travelers only)

  • Query: yearsInBusiness Integer

    Number of Years the Company has been in business

  • Query: yearsOfCreditHistory Integer

    Number of Years the Company has had credit history

  • Query: numberOfEmployees Integer

    Number of Employees at Company

  • Query: numberOfClaims Integer

    Number of Insurance Claims the Company has ever filed
