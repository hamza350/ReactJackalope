Getting Started

NinjaRater Rest APIs

To get your API key, please call us at 1 (866) 764-6610 or contact us by email at sales@ninjarater.com

Download NinjaRaterAPI.swagger.json

Import it in Swagger Editor for live testing of NinjaRater APIs.









Click the left arrow on the middle divider to hide the RAML code and make it full screen.

Click on the green Authenticate area and Enter your API Key

Click on the tag of an operation you want to try.

Click on Try this operation. Default values are filled in for you.



Click on Send Request.

JSON response appears below.









Enjoy!
